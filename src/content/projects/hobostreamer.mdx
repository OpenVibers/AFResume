A community-first live streaming platform built from scratch. Connect OBS/ffmpeg or stream directly from the browser, then restream simultaneously to Twitch, YouTube, Kick, and any RTMP endpoint from a single dashboard.

Features: real-time chat with custom emotes, automatic clip detection, auto-recorded VODs, built-in HoboQuest game for viewers, sitewide global chat, voice channels, pastes & file sharing with syntax highlighting, full stream analytics, and comprehensive moderation tools. Every line of code is public on GitHub.

Servers went offline in May 2026 when the funding ran out during a job search. The architecture is sound, the codebase is complete, and it's all open source — ready to self-host or fork.
