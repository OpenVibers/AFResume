A conversational analytics assistant for Amazon ICQA. The bot ingests the
warehouse data dictionary, builds a schema graph, and uses Anthropic's Claude
models to answer free-form questions with **runnable, table-aware SQL** plus an
explanation of the join path it chose.

**Highlights**

- Auto-discovers tables, foreign keys, and column semantics so analysts don't
  have to remember 200+ table names.
- Streams partial SQL with inline rationale, then offers one-click execute and
  result preview.
- Hardened prompt-injection guards on tool calls; nothing executes without a
  human confirm.
